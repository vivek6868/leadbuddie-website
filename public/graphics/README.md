# Graphics Folder

This folder contains graphics and images for the LeadBuddie website.

## Structure

- `hero/` - Hero section graphics and animations
- `icons/` - Custom icons and SVG graphics
- `partners/` - Partner logos (WhatsApp Business, etc.)
- `metrics/` - Metric badges and graphics

## Usage

All graphics in this folder can be referenced from the public directory:
- `/graphics/hero/example.png`
- `/graphics/icons/example.svg`
- `/graphics/partners/whatsapp.png`

## Note

This is a new graphics folder separate from the existing `/images` folder.
